Emmet Abbreviations for Web Development

HTML Comment

Emmet is a powerful toolkit for web development that speeds up HTML and CSS writing. With its shorthand abbreviations, you can generate snippets of code quickly. Here are some common Emmet abbreviations and how they expand:

DownloadEmmet Cheat Sheet

1. Emmet (Built-In) Description: Provides built-in support for Emmet, speeding up HTML and CSS workflow.
Benefits: Shortcut-driven, reducing the amount of typing required to write boilerplate code. Install: Built into VSCode; no installation required.

2. Prettier - Code Formatter Description: Automatically formats your code based on defined style guides. Benefits: Ensures consistent styling across projects. Install: Search for Prettier - Code Formatter in the Extensions Marketplace.

3. Live Server Description: Launches a local development server with live reload functionality. Benefits: Reflects changes immediately without needing to refresh the browser. Install: Search for Live Server in the Extensions Marketplace. Usage: Right-click your index.html file and select Open with Live Server.

Using Emmet Abbreviations to create simple HTML Elements

div + Tab: Creates a div element.

h1{Heading Text}: Creates an h1 tag with the specified text.

Heading Text

div.className: Creates a div with the specified class.

#idName: Creates a div with an ID.

ul>li*3: Creates an unordered list with 3 list items.

a[href="https://example.com"]: Generates a link with an href attribute.

img[src="image.jpg" alt="Image description"]: Generates an image tag with src and alt attributes.

nav>ul>li*3>a: Generates a navigation structure with 3 list items containing anchor tags.

.container>header.header>nav.nav>ul>li*3>a[href="#"]: Creates a nested structure for a common layout.

CSS Emmet Abbreviations

m10: Adds margin of 10px.

margin: 10px;

p20-10: Adds padding of 20px top/bottom and 10px left/right.

padding: 20px 10px;

pos-r: Generates position: relative;

position: relative;

bd1-s#000: Generates border: 1px solid #000;

border: 1px solid #000;
