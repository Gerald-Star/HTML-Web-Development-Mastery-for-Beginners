Block Elements

Block-level elements always start on a new line and take up the full width available (stretches out to the left and right as far as it can).

Block-level elements can contain inline elements and other block-level elements.

Block elements are usually structural elements that define the layout of the page.

Block elements does not allow other elements to sit behind

Block elements have top and bottom margin unlike inline element

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary.

Inline elements can only contain other inline elements and data.

Inline elements are usually text elements that define the content of the page.

Inline elements do not have top and bottom margin.

Inline elements allows other inline elements to sit behind

Inline elements can have left and right margin and padding but not top and bottom
