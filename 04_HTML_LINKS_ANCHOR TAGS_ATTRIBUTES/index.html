Absolute Link

An absolute link is a link that points to a specific URL. Absolute links are used when linking to pages outside your website.

The target Attribute in Links

The target attribute specifies where to open the linked document. The target attribute can have one of the following values:

  • _blank attribute opens the document in a new external window or tab
  • _self attribute opens the document in the same window/tab as it was clicked
  • _parent attribute opens the document in the parent frame
  • _top attribute opens the document in the full body of the window
