HTML FORMATTING ELEMENTS

HTML Formatting Elements are used to format the text and content of a webpage. These elements are used to create headings, paragraphs, lists, and more. The following are some of the most commonly used HTML Formatting Elements:

  • Deleted text: The del tag is used to create deleted text on a webpage. The del tag is used to display text that has been deleted.
