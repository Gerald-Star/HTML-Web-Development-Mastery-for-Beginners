The HTML Picture Element

The HTML picture element contains zero or more source elements and one img element to offer alternative versions of an image for different display/device scenarios.

Picture Element Example

Each source element must have a srcset attribute and a media attribute to specify the image file and the media condition (screen width) for each image. The img element must have an alt attribute to specify an alternate text for an image, if the image for some reason cannot be displayed. The img element have to be specified as the last child of the picture element. The browser will use the first source element with a matching media condition. The browser uses the img element if none of the source media conditions are met. The style attribute is used to specify the width of the image. The value auto means that the browser will calculate the width of the image.
